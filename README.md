# BlokusGUI プロジェクト
- BlokusのGUI実装例
- Visual C# 2022

# 実装
- 不足ピース
- ピースの回転
- 各プレイヤーの開始点設定と表示
- 配置ルールの検証
- 勝敗判定と表示
- 対戦終了後に最初のメニューに戻る
- ピースを置く前ゲーム終了の例外処理
- ボタンでゲーム終了
- リアルタイムポイント記録
